This module uses the HPMC library for iso-surface extraction and rendering:
"HPMC is a small OpenGL/C/C++-library that extractes iso-surfaces of volumetric data directly on the GPU."
http://sintef.org/Projectweb/Heterogeneous-Computing/Research-Topics/Marching-Cubes-using-Histogram-Pyramids/

Win32:
  The included HPMC library has been built for MS Visual Studio 2008.
  If you use another compiler, it might be necessary to re-build it.
